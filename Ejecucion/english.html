AP5.AA1.E5. HTML5 Y CSS3 QUIZ

QUIZ

HTML5 y CSS3 son los 2 lenguajes de programación más utilizados a la hora de generar páginas web.

Tanto HTML como CSS son lenguajes bastante simples. Actualmente las últimas versiones de

estos dos lenguajes; el HTML5 y el CSS3, ofrece al usuario múltiples posibilidades de diseño.

Con estos lenguajes se pueden realizar las siguientes funciones: añadir texto a un sitio web,

construir un menú de navegación, añadir imágenes, efectos gráficos y también funciona para

crear diversos filmes animados, aplicar efectos visuales y aplicar modelos de impresión 3D.

En esta evaluación el aprendiz responderá 10 preguntas relacionadas a estos 2 lenguajes.
